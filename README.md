golly
=====

it's my pet that is a my dreamed OS!

开发环境：
win7 


本系统是按人类世界种子发育的不同阶段来实现。
主要分为以下几个阶段：

一、系统环境准备阶段，主要分为以下两个阶段：

1、引导程序：
    单纯的加载系统萌发条件。
2、系统萌发条件程序：
    为种子的播种提供条件，并开始播种

二、种子萌发阶段：
    主要是吸收系统硬件环境资源，开始发育。

三、种子发育阶段：
    主要是形成特化器官和组织，为提供系统服务作准备。

四、golly诞生：
    主要是一个低智能的内核系统，以一个简单的shell为贫道提供服务。
